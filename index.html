Galería de Pruebas CSS-in-JS

Selecciona una prueba para verificar visualmente las capacidades del motor de estilos.

01. Aislamiento Básico

Verifica que los estilos no se filtren entre componentes idénticos.

02. Anidamiento (Nesting)

Pseudo-clases (:hover), pseudo-elementos (::after) y selectores anidados.

03. Animaciones Aisladas

Keyframes con el mismo nombre en diferentes componentes no colisionan.

04. Diseño Responsivo

Media queries anidados para adaptar el diseño según el ancho.

05. Variables y Fallbacks

Variables CSS (case-sensitive) y arrays de valores de respaldo.

06. Escape Global

Uso de :global() para afectar estilos fuera del componente.

07. Pseudo-elementos y Reglas-At

Pruebas de pseudo-elementos y reglas como @media y @supports.

08. Recolección de Basura

Verificación interactiva de la eliminación automática de estilos no usados.

09. Combinadores y Reglas Complejas

Selectores avanzados (>, +, ~), atributos y reglas @supports.

10. Selectores Complejos

Manejo correcto de comas dentro de atributos y funciones como :is().

11. CSS Moderno

Pruebas de características modernas como @layer, @container y :has().

12. Rendimiento

Benchmark de creación masiva y recolección de basura.
